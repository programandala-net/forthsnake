== Serpentino
:author: Marcos Cruz (programandala.net)
:revdate: 2017-11-24

A text-based snake game under development written in
http://forth-standard.org[Forth,role="external"] for
http://gnu.org/software/gforth[Gforth,role="external"] (Gforth 0.7.9
required).
